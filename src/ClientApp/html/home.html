Welcome to the Node Agent Environment (NAE) project

NAE is an multiagent environment written in Node.js. It consists of the following separate projects:

  • AgentCenter
  • Broker

Be sure to check out the following pages:

  • The agent runner page, where you can create and send custom messages to all of our working agents.
  • The examples page, where you can activate preconfigured examples and with just a few clicks create and activate our agents.
  • Our GitHub repository, where you can look at our source code and get all the published information about this project.
